👋 Welcome to the Chatting App Back-End

This page will help you use the OctoberCMS admin panel easily — even without programming knowledge.

🔐 How to Access the Admin Panel

  1. Go to /cms in your browser

🧭 What Can You Do in the Admin Panel?

As a regular user, you should not modify CMS Pages or Media files.

Focus on:

  • 🧑 Managing Users
  • 💬 Creating and viewing Conversations
  • 📩 Sending and reviewing Messages
  • 😄 Managing Reactions

All of these actions can be done using the backend interface via Controllers.

🛠 Recommended Tools (Optional)

If you want to inspect the database directly (not required), you can use:

  • TablePlus
  • phpMyAdmin (comes with MAMP)

📚 Want to Learn More?

Check out the official documentation here: October CMS 3.x Documentation

You're ready to go! 🎉
